CSC 309 User Acceptance Testing



Acceptance Test Plan By Jordan Tang

The tabular format of an acceptance test plan is the following:

Case No. User Inputs and Commands Expected Output Remarks                   Results                  
1 Run application. From the taskbar, click "editor". There should be no response from the application. This makes sure that no user cannot edit without declaring creating a new lecture Pass
2   Click on files, then click create.   There should be no response from the application.   This sets the application to "create lecture" mode.   Pass
3   Click on editor on the taskbar.   The editor window should now appear.   This makes sure the editor appears after setting to edit mode.   Pass
4   On the editor window, select "Slide 1".   Slide 1 should now be highlighted on the editor window.   This makes sure a slide is selected to be edited.   Pass
5   Click on the HTML button the editor window that appeared.   The HTML slide editor window should appear.   This makes sure the HTML editor appears.   Pass
6   Type in html tags in the text area, along with any html code. Click generate slide to show the html.   The html should be rendered on the rightside of the editor.   This makes sure the editor is properly showing a preview of rendered HTML   Pass
7   Close the HTML editor box. The HTML box should dissappear.   The presentation area should display the HTML.   Pass  
8   On the editor window, click the "+" button   Slide 2 should appear below Slide 1. Slide 2 should also appear on the presentation bar.   This makes sure a slide is successfully being added   Pass
9   On the editor window, click the "-" button on Slide 2   Slide 2 should dissappear.   This makes sure a slide is successfully being deleted   Pass
10   Click the "+" button to add a new slide on editor window. Click Slide 2.   Slide 2 should be highlighted on the list.   This makes sure the Slide is properly being selected.  
11   Click the Slide 2 on the presentation.   Slide 2 should be selected.   This tests that slide 2 is also selectable in presentation.   Pass
12   Open the html editor. Close the HTML window with a blank text area.   The slide on presentation should be blank.   This tests that html is being rendered even with a blank window.   Pass

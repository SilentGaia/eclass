E-Classroom Tool Integration Test Plan


After the classes in each package are independently tested with taskbar and classroom stubs, perform integration testing in the following phases:

  1. Integrate the presentation package with the classroom package and rerun the class tests in the presentation package.

  2. Add the layers package to the integrated presentation + classroom packages; rerun the class tests in the presentation and layers packages.

  3. Add the taskbar package to the integrated presentation + layers + classroom; rerun the class tests in the presentation, layers, and taskbar packages.

  4. Integrate the classroom and classroom.rmi packages and rerun the class tests in the classroom package.

  5. Add the classroom.rmi package to the integrated presentation + layers + taskbar + classroom; rerun the class tests in the presentation, layers, and taskbar packages.

  6. Add the drawingtools package to the integrated presentation + layers + taskbar + classroom + rmi; rerun the class tests in the presentation, layers, taskbar, and drawingtools packages.

  7. Add the roster package to the integrated presentation + layers + taskbar + drawingtools + classroom + rmi; rerun the class tests in the presentation, layers, taskbar, drawingtools, and roster packages.

  8. Add the chat package to the integrated presentation + layers + taskbar + drawingtools + roster + classroom + rmi; rerun class tests in the presentation, layers, taskbar, drawingtools, roster, and chat packages.

  9. Add the top-level eclasstool classes to the integrated packages and run the top-level class tests, which in turn run the class tests in each package.

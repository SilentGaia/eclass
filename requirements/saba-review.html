1.5.2. Saba

Saba is a virtual classroom software that allows people to learn in a highly interactive environment. The main purpose of Sabaâ€™s virtual classroom is to provide the most accessible online collaboration experience. No matter where an individual is, the ease to enter the virtual classroom and become well educated and trained is one of the key benefits of using Saba. It tries to offer a superior user experience, being easy to learn and easy to use. The features provided by Saba will be taken into consideration when developing the Electronic Classroom tool.

Good Features:

  • Video conferencing allows for face-to-face interactions
  • Voice using VoIP (Voice over IP)
  • Easy class management
  • Agenda Builder
  • Pre-session mode (Allows one to check out lecture before its given)
  • Responses through emoticons
  • Breakup rooms (Divides classes into smaller groups to work on single projects)
  • Content available after lecture
  • Attendance and participation check in
  • Mobile online classes via iPhones or iPads
  • Real time online classroom polls

Bad Features:

  • Attendance cap in the classroom
  • Chat is public to the whole class

Missing Features:

  • Layers for lectures

1.5.5. Google Slides

Google Slides is a presentation tool which allows users to create and edit presentation slides online while collaborating with other users live. The application is for multi-platform use, as it is available as several web applications, Chrome apps that work offline, and mobile apps for Android and iOS.

Overall, Google Slides provides an acceptable level of presentation functionality but it does not have the full functionality of an electronic classroom.

Good Features:

  • all the basic presentation functionality is provided
  • compatible with Microsoft Office
  • available for web applications, Chrome, Android, and iOS
  • good integration with related tools (Google Drive suite)
  • simple
  • ease of collaboration
  • frequent product updates

Bad Features:

  • not primarily an electronic classroom tool, so lacks advanced features desired for the tool proposed in these requirements
  • lacks variety of customization tools

Missing Features:

  • specific administrator-to-student abilities
  • sophisticated layer options
  • private viewing options
  • no apparent support for chatting features

2.9. Details of File and Edit Operations

The scenarios in this section show a user how to do a variety of program administration tasks, including task bar capabilities and short cut keys for file handling.

Opening lecture files is detailed in the subsections of section 2.8. To save lecture files, the user presses the floppy disk icon at the bottom right of the taskbar window. The user saves the lecture file either locally on his/her own computer or in the database outsourced from another program provided by the E-Classroom tool.

The information stored in the lecture file is a compilation of a lecture presentation with saved drawing objects and overlays from the user who is saving that specific lecture file.

2.9.1. Shortcuts and Hotkeys

Another way users do 'Edit' commands in the EClassroom tool is to use keyboard shortcut keys. The following key commands are included:

  • 'Save': CTRL-s, CTRL-S
  • 'Undo': CTRL-z, CTRL-Z
  • 'Redo': CTRL-y, CTRL-y
  • 'Toggle Mode': CTRL-l, CTRL-L

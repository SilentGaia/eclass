1.5.6. Invision Power IP.Chat

Ivision Power’s IP.Chat is a chatbox that is integrated with the forums that Invision Power provides, and can be used for a general chatbox, and separate chatrooms for user to user chats.

Overall, IP.Chat is an application that provides decent functionality as a chatbox, but the tool has to be integrated with the overall forums.

Good Features:

  • Has public chatroom for entire group.
  • Also has one-to-one private chats
  • Allows censoring of words.
  • Allows users to have moderator status in order to delete messages in the public chatroom, and to kick people out of the chatbox.
  • Has optimum availability.
  • Shows list of users online.
  • Ability to post pictures, and modify font.

Bad Features:

  • Kicks out people who are inactive for too long.
  • Has sounds when people enter the room (you can mute).
  • Requires to be integrated with IP board forums.

Missing Features:

  • Ability to run as it’s own program

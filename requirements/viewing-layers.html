2.8.4. Viewing Overlays

After a presentation is done and instructor, or the students, themselves wants to view over the slides, including the layers added during the presentation, can be viewed in the Layers window and the corresponding slide/layer will also show up on the Slide window.

To look over a lecture's slide/layers, the user must first open the file as section 2.8.2 has demonstrated. Then, all the slides and layers will show up in the Layers window as shown in Figure 112.

Figure 112: HTML Edit - Main view.


From there, the user can use the scroll bar on the right side of the window to navigate through the lecture or the user can use the navigation bar provided in the Slide window to navigate through lecture.

2.2.4. Details of Lecture Format

The tools provided in this Electronic Classroom to create a lecture are simple and straight forward. Users have the ability to further customize their presentations. When the user selects the HTML button in the top right corner, the HTML editor will appear as shown in Figure 26.

In order to edit the HTML format of the lecture, the user needs to open up the 'Insert Lecture' window and select on a lecture slide. On the window, there is a button labeled HTML on the right top corner. When the user presses the "HTML" button, the system will display the editor as shown in Figure 26.

Figure 26: HTML Edit - Main view.


Once the user presses on the HTML button, the window will be split into two parts. The left side is the typical 'Insert Lecture' screen and on the right side is the HTML format of that specific lecture slide. To edit the HTML, the user types directly in the box. In Figure 27, a bullet point is added into the HTML format screen.

Figure 27: Editing HTML screen.
As the user might have noticed, the normal screen on the left does not show the added bullet point. This is because the user must click on the refresh button at the top right corner of the right screen. Once the user presses the refresh button, the right screen should match up with the left screen and vic versa. Figure 28 shows the refreshed screen, after adding the bullet point in the HTML screen.


Figure 28: Editing Lecture Slide - Refreshed Screen.



Here are some HTML tags specific to our program as shown in the table below. Anything that is not in the HTML table will be rendered to W3C formatting.


HTML Format Table

HTML Tag Electronic Class Interpretation
<hr> slide break
<ol type = I|A|1|a|i, IS_EXPANDED = TRUE|FALSE> Start roman numeral topics
<ol type = 1|1.1|1.1.1, IS_EXPANDED = TRUE|FALSE> Start decimal topics
<ol type = disc|circle|square, IS_EXPANDED = TRUE|FALSE> Start bullet point topics
<P hidden> Don't show during lecture, but show offline with 'Show Details'
<li> one topic
</ol> End numbered topics

1.5.4. Heux

Heux is an interactive e-learning software built for the classroom setting. The main purpose of the software is to improve current teaching methods as well as increase interaction between teacher and students. The main functionality of Heux centers around teachers being able to upload PowerPoints and PDF Slides into interactive whiteboard slides, which then get saved into a lesson plans for students to access both in and out of class. Students are also given the capability to write and edit on lesson plans and post on forums for real-time communication with fellow classmates. With the successful integration of these key features, Heux demonstrates similar requirements to the E-Classroom.

Overall, Heux offers important features that are also represented in the E-Classroom.

Good Features:

  • effective Cloud storage of lessons
  • simple and intuitive layout
  • upload several types of files
  • re-use lesson plans from past
  • converts PowerPoint and PDF files into whiteboard slides for writing and editing
  • can conduct polls for students
  • real-time forum for discussions
  • real-time monitoring of students
  • can lock students’ screens to command attention
  • list of students participating in lesson is readily available
  • students can be given control of lesson
  • slides can be created during the lesson
  • multi-platform: iOS, Android, Windows 8
  • can create lessons without another tool
  • universally accessible
  • not limited to classroom machines
  • includes simple writing and editing tools

Bad Features:

  • tag submission that awaits students’ responses before proceeding can be interruptive
  • background includes only pre-defined templates
  • my device can be locked if connected during lesson and instructor locks screen

Missing Features:

  • mini-administrators for certain tasks like editing slides, drawing, taking notes, etc.
  • grading system
  • calendar for events and due dates
  • chat room
  • pinging instructor with questions during lesson
